## ✨ Features

- 🕐 **Real-time display** of hours, minutes, and seconds
- 🌗 **AM/PM format** indicator
- 🔄 **Auto-updates** every second
- 📱 **Responsive design** — works on desktop and mobile
- 🎨 **Minimalist UI** with clean layout and subtle effects

## 🚀 Getting Started

1. **Clone** or download the project files

   ```bash
   git clone https://github.com/MuhammadAfnanAkhta/Digital-Clock.git
   ```

2. **Open** `index.html` in any modern web browser

   ```bash
   cd Digital-Clock
   open index.html
   ```

3. The clock starts displaying the current time **immediately** ✅

> No server, build tools, or additional setup required — it's a pure client-side application.

---

## 🎨 Customization

**Appearance** — edit `style.css`:
- Change the background image or color
- Adjust font sizes and colors
- Modify the clock container's size and shadows

**Functionality** — edit `script.js`:
- Alter the time format (12h / 24h)
- Change the update frequency
- Add date display or timezone support
